# Spring MVC Lab
Halo! Ini adalah project Spring MVC Lab yang saya kerjakan sebagai bagian dari tugas kuliah.  
Di project ini, saya belajar tentang Spring Boot + Thymeleaf, termasuk eksperimen-eksperimen berikut:

## 🎯 Tujuan Project
Tujuan dari lab ini adalah:
1. Memahami cara kerja Spring MVC (Controller, Model, View)
2. Mempelajari perbedaan antara `@Controller` dan `@RestController`
3. Menggunakan Thymeleaf untuk membuat halaman HTML dinamis
4. Mengelola static resources seperti CSS dan gambar
5. Membuat fitur tambahan seperti fragment, breadcrumb, dan halaman statistik

## 💡 Catatan Pribadi
- Saya belajar banyak tentang bagaimana Spring Boot memisahkan controller, model, dan view.
- Eksperimen fragment dan static resources membantu saya memahami bagaimana Thymeleaf memproses template dan CSS.
- Screenshot di folder `screenshots/` saya kumpulkan sebagai bukti eksperimen berhasil.
